# Body Shape Coach Telegram Bot

A fitness coaching Telegram bot powered by OpenAI, deployed on Cloudflare Workers.

## Features
- Personalized workout plans
- Nutrition advice
- AI-powered fitness coaching
- Multi-language support (English)

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Deploy to Cloudflare: `npm run deploy`
4. Set Telegram webhook

## Deployment

Deployed on Cloudflare Workers at: `https://body-shape-coach-bot.sportschatbot01.workers.dev`
